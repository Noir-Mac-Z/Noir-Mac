## Overview

Noir Professional Suite is a complete dark browsing platform with theme customization, allowlisting, and sync for comprehensive dark mode control.

Custom theme colors replace the automatic dark palette with user-specified colors for background, text, and accent elements, creating a personalized dark browsing appearance.

Allowlist and blocklist control which websites receive dark mode treatment, enabling dark mode everywhere except specified sites or only on specified sites depending on preference.

iCloud sync keeps per-site customizations and preferences consistent across all Apple devices running Safari.

Noir Professional Suite is the dark mode platform for Safari users who want comprehensive control over dark browsing with the customization and sync that consistent cross-device dark mode requires.

Noir is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Noir processes operations quietly behind the scenes.
